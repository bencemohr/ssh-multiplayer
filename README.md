# Project MITS - Multiplayer

The home repository for the MITS Multiplayer API

## Setup
1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Database**:
    Ensure PostgreSQL is running (via Docker).
3.  **Seed Admin User**:
    ```bash
    node seed-admin.js
    ```
    Creates user `admin` with password `password123`.
